# Project: OpenFarm Survival

## ⚠️ Required Dependencies
This project uses the **Godot Jolt** physics engine for stable vehicle towing and implement physics. Because of the large binary sizes, Jolt is not included in this repository.

**To run this game:**
1. Clone this repository and open the project in Godot 4. *(Note: Godot will throw some red errors in the console on the first launch. Ignore them!)*
2. In the Godot Editor, click **AssetLib** at the top.
3. Search for **Godot Jolt**, download it, and install it.
4. Restart Godot. The physics engine will automatically hook up, and the game is ready to play.
